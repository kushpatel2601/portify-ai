## Features

- AI-powered storytelling
- ATS score calculation
- Dynamic portfolio generation

## Tech Stack
- Next.js
- FastAPI
- Groq LLM
- GitHub API

## Run Locally

### Backend
cd backend
uvicorn main:app --reload

### Frontend
cd frontend
npm install
npm run dev
